Metodos Numericos *(MAT-205)*

En esta materia tipo calculo se enseño a buscar la raiz de funciones en un plano cartesiano, se utilizo diferentes metodos, como por ejemplo 
la serie de taylor, biseccion, falsa posicion, newton rapgson, secante, gauss simple, gauss jordan, factorizacion LU, factorizacion LU inversa

El proyecto para esta materia consistia en programar cada uno de estos metodos.
